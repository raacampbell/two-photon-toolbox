Import

  • addDatData - Add Trigger Sync analogue .dat data to twoPhoton object
  • addStimParams - Add stimulus parameters to data structure
  • generateDFFobject - Convert imported data structure into the twoPhoton data object
  • importFluoView - Import a tiff generated by FluoView
  • importFluoViewBatch - Import all FluoView tiff files in the current directory
  • importPVtree - Import data from multiple Prairie view data directories
  • prairieXML_2_Matlab - Import PrairieView XML and image-stacks to MATLAB
  • readPRM - Import Trigger Sync PRM file
  • readPVdat - Import Prairie view .dat file
  • twoPImportBatch - Import and pre-process all PraireView data in current directory
  • importScanImage - Import a tiff generated by ScanImage
  • importSItree - Import data from multiple ScanImage directories

Pre-Processing

  • filterImageStack - Smooth frames to make the look prettier
  • doPhotoBleachCorrection - Conduct photo-bleach correction on twoPhotonObject

Motion Correction

  • alignStack - Apply 2-D translation correction to animage-stack
  • alignRepeats - Conduct translation correction across trials
  • apply_ffttrans - Apply fft-based translation correction
  • apply_demon - Apply non-linear, liquid, demon registration
  • apply_elastix - Apply elastix registrations
  • apply_gkerr - Apply Greenberg & Kerr 2-photon correction
  • apply_symmetric - Apply GPU-based non-rigid correction
  • regParams - add, remove and modify the registration parameters
  • visualiseDrift - Assess the degree of motion across repeats with cross-correlation
  • visualiseDriftMovie - Assess the degree of motion across repeats with a movie

ROIs

  • ROI_add - Append a new ROI to the ROI structure of the twoPhoton object
  • ROI_autodetectFancy - Use thresh_tool to perform a semi-automated detection of the main ROI
  • ROI_batch - Determine the main ROI for each stimulus presentation
  • selectROIs - Select neuronal cell bodies with a GUI
  • nNeurons2p - Summarise number of responding neurons, etc
  • showPhotoBleachFit - Plot time-series before and after photo-bleach correction

Basic Neuronal Statistics & Plotting

  • odour3Dplot - 3-D PC plot of each stimulus
  • odourDendrogram - dendrogram showing response relatedness
  • whichROIsAreSignificant - Show how many times each ROI responded significantly
  • plotROIstats - Plot neuronal response traces ordered by signifiance

Data Display

  • cleanMeanDFF - Calculate a reduced-noise mean dF/F during the response
  • colorDepth - Produced colour-coded depth plot of a z-stack
  • colorResponse - colour-coded maximum intensity projection
  • imageStackCorr - plots correlation between 1st and later frames from one trial
  • Kalman_Stack_Filter - Apply a Kalman filter to the time dimension of an image stack
  • kcMaskPlot - Show activation of each cell in the mask using different colours.
  • overlayDFFandBaseline - overlay of significant dF/F onto baseline image and plot
  • overlayDFFs - Overlay significant dF/F pixels from two recordings and plot
  • playMovie - Play a movie of an imageStack
  • responseTimeCourse - Plot response time course of an ROI
  • roiTimeCourse - mean pixel intensity over time from a defined ROI
  • showSelectedKCs - Overlay selected KCs onto a mean image of the stack

Summarize Recording Session

  • generateRecordingReport - Make an HTML report of one experiment
  • responseMagByPresentation - Response magntitude of each trial
  • responseTimeCourseByOdour - response time courses grouped by odour

Utility Functions

  • data2libsvm - convert twoPhoton data to libsvm format
  • getOdourNames - Summarise stimuli used in one experiment
  • load3Dtiff - Load multi-image tiff from disk
  • mat2im - mat2im - convert to rgb image
  • meanFromTif - Calculate mean of a multi-image tiff
  • nNeurons2p - Return neural summary statistics
  • patch2blender - Convert image stack to blender format
  • responsePeriodFrames - Determine which frames fall into the response period
  • save3Dtiff - Save 3-D grayscale matrix 'imageStack' as 3-D tiff 'fname'
  • tiffFrames - Iteratively find the number of slices in a tiff
  • updateDataDir - Update path to data directory
  • zExplorer - Scroll through a z-stack with a slider
